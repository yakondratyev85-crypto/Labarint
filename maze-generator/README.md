# Генератор лабиринтов SVG

Это полностью локальное веб-приложение для генерации лабиринтов разных типов и сложности с экспортом в чистый векторный SVG.

Проект не использует npm, Vite, React, Vue, Angular, TypeScript, Webpack, серверы или внешние библиотеки. Всё работает в обычном браузере из статических файлов.

## Как запустить

1. Скачайте репозиторий, например через GitHub Desktop.
2. Откройте папку `maze-generator`.
3. Дважды кликните по файлу `index.html`.
4. Генератор откроется в браузере и сразу построит первый лабиринт.

Сервер запускать не нужно.

## Возможности

- Генерация лабиринтов по выбранным настройкам.
- Предпросмотр результата прямо на странице.
- Повторяемая генерация через seed.
- Кнопка `Random Seed` для быстрого случайного seed.
- 5 уровней сложности: `Very Easy`, `Easy`, `Medium`, `Hard`, `Expert`.
- Настройка ширины, высоты и толщины стен.
- Поиск решения через BFS.
- Показ и скрытие решения.
- Экспорт SVG без решения.
- Экспорт SVG с решением.
- Информационная строка с типом, сложностью, seed, размером и длиной решения.

## Поддерживаемые типы лабиринтов

1. **Rectangular Maze** — классический прямоугольный лабиринт на квадратной сетке. Генерируется алгоритмом DFS / Recursive Backtracking. Вход находится слева сверху, выход — справа снизу.
2. **Circular Maze** — круговой лабиринт из концентрических колец и радиальных перегородок. Стены строятся SVG-элементами `path`, `circle` и `line`.
3. **Hex Maze** — лабиринт на шестиугольной сетке, где у каждой клетки до 6 направлений.
4. **Room / Island Maze** — прямоугольный лабиринт с дополнительными комнатами, открытыми зонами, островами и проходами. Путь от старта до финиша сохраняется всегда.
5. **Shape Maze** — лабиринт внутри формы. Поддерживаются формы `circle`, `heart` и `star`; клетки вне формы не отображаются.

## Как скачать SVG

1. Настройте тип лабиринта, сложность, размер, толщину стен и seed.
2. Нажмите `Generate`.
3. Для файла без решения нажмите `Download SVG`.
4. Для файла с красной линией решения нажмите `Download SVG with Solution`.

Экспортируемый SVG является чистым вектором. В нём нет PNG, JPG, base64, canvas bitmap, `foreignObject` или растровых вставок. SVG содержит слои:

```xml
<g id="maze-walls">...</g>
<g id="maze-solution">...</g>
<g id="maze-labels">...</g>
```

Файлы можно открывать в браузере, Figma, Illustrator и Inkscape.

## Seed и повторяемость

При одинаковом seed и одинаковых настройках генератор создаёт одинаковый лабиринт. Если изменить seed, структура лабиринта изменится. Для детерминированной генерации используется seeded random на основе `mulberry32`.
